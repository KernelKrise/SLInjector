# SLInjector

Simple Library Injector

## Description

View [description file](./DESCRIPTION.md)

## Compile

Run the following command in repository root:
```
make
```

## Usage

```
./bin/slinjector <pid> <shared_library_path>
```

## Compile example libs

Run the following command in repository root:
```
make libs
```

## Supported OS

Linux

# Supported architecture

amd64
